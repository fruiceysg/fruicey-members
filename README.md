# Fruicey Members App

Membership tracking app for Fruicey — built with plain HTML, no backend needed.

## Files
- `index.html` — Staff admin panel (dashboard, members, transactions, register, backup)
- `portal.html` — Customer self-service portal (check points & tier by phone/email)

## Setup on GitHub Pages
See instructions below.

## Features
- Gold / Platinum / Diamond tiers with 2-month renewal
- Automatic tier downgrade logic
- Birthday treat tracking
- Points (1 pt per $1 spent)
- Tier-based discount calculator
- Export/import data as JSON backup
- Export members as CSV

## Data Storage
Data is saved in the browser's localStorage on whichever device you use.
Use the **Backup tab** to export regularly and keep a copy safe.
